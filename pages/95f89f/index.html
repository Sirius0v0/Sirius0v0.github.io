搭建带有CDN加速的免费图床

# 搭建带有CDN加速的免费图床

# 准备

  • [x] github
  • [x] PicGo (opens new window)
  • [x] jsDelivr (opens new window)

# 搭建

  • github中新建一个图片仓库（Sirius0v0/image_store）
  • 打开Settings - Developer settings - Personal access tokens，点击Generate new token
  • 填写token，勾选repo，==注意名称要与图片仓库同名（即image_store）==
  • 将生成的token填入PicGo的github图床设置中，并根据自己个人习惯设置快捷上传键

# 使用

  • 截图
  • Ctrl + Shift + P
  • Ctrl + v到文档
